# THP Call Reporting Responsibilities

Mobile web app for radiologists on call. Pick your rotation, see your responsibilities.

Live URL: **https://lillyteng.github.io/thp-call-rr/**

## Files

- `index.html` — self-contained app (offline-capable via service worker)
- `service-worker.js` — PWA offline cache
- `THP-Call-RR.pdf` — tightened by-rotation reference PDF
- `THP-Call-Reporting-Responsibilities-source.pdf` — Ivan Diamond's original source document
- `.nojekyll` — disables GitHub Pages' Jekyll processing

## How the app is rebuilt

The app is generated by `build_call_app.py` in the workspace `THP Call Reporting/_scripts/` folder. Do not edit `index.html` directly.
